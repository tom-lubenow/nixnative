# Default Options

### `native.defaults.compileFlags`

Default compile flags.

**Type:** `list of string`

**Default:** `[]`


### `native.defaults.defines`

Default preprocessor defines.

**Type:** `list of (string or (attribute set))`

**Default:** `[]`


### `native.defaults.includeDirs`

Default include directories.

**Type:** `list of (absolute path or string or (attribute set))`

**Default:** `[]`


### `native.defaults.languageFlags`

Default per-language compile flags.

**Type:** `attribute set of list of string`

**Default:** `{}`


### `native.defaults.libraries`

Default libraries.

**Type:** `list of (string or absolute path or (submodule) or (attribute set) or (attribute set))`

**Default:** `[]`


### `native.defaults.linkFlags`

Default link flags.

**Type:** `list of string`

**Default:** `[]`


### `native.defaults.publicCompileFlags`

Default public compile flags for libraries.

**Type:** `list of string`

**Default:** `[]`


### `native.defaults.publicDefines`

Default public defines for libraries.

**Type:** `list of (string or (attribute set))`

**Default:** `[]`


### `native.defaults.publicIncludeDirs`

Default public include dirs for libraries.

**Type:** `list of (absolute path or string or (attribute set))`

**Default:** `[]`


### `native.defaults.publicLinkFlags`

Default public link flags for libraries.

**Type:** `list of (string or absolute path)`

**Default:** `[]`


### `native.defaults.tools`

Default tools.

**Type:** `list of (attribute set)`

**Default:** `[]`
